# CPP-EDITOR
一个轻量级、高效的 C++ 编辑器！使用 Python 打造！
此编辑器基于 Python 制作，Nuitka 打包，
支持 Microsoft Windows 10-11 系统。

## 此编辑器的亮点？
* 此编辑器使用精简的 GNU-GCC 编译器，自带控制台调试器。
* 支持 __在临时目录运行（不保存编译运行）__。__自带 Online Judge (OJ) 系统__，允许编辑题目、自动判题。
* 支持深色模式、浅色模式跟随系统切换。
* 支持代码补全、括号匹配、智能缩进、代码高亮、行数显示、批量选择、自动换行等。
* 界面简洁，无多余控件，只有代码编辑器和菜单栏。

## 此编辑器的用法？
快捷按键：
|按键|功能|
|F1|显示帮助。|
|F3|显示/隐藏 OJ 题目显示。|
|F4|自动判题。|
|F5|F11：编译并运行（不需要保存）。|
|F9|编译（需要保存）。|
|F10|运行（需要保存）。|
|Alt+1|插入控制台头文件模板。|
|Alt+2|插入 Windows API 窗口模板。|
|Ctrl+MouseWheel|缩放字体。|
>更多快捷按键在程序中。
